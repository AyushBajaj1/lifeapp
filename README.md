# Lifeapp

**A task management web application focused on organization and self-improvement.**  
The app currently provides a fully functional task manager with hierarchical organization.

## Overview

Lifeapp is a React-based task manager designed to help users organize tasks using **categories and nested subcategories**. The project focuses on clarity, structure, and extensibility rather than feature bloat.

At its current stage, Lifeapp is a frontend application with core task management functionality implemented.

## Features

- Create, edit, and delete tasks  
- Organize tasks into categories  
- Create subcategories within categories  
- Clean, responsive UI built with **Tailwind CSS**  
- Component-based architecture using **React**

## Tech Stack

- **React** – UI and state management  
- **Tailwind CSS** – Styling and layout  
- **JavaScript (ES6+)**
